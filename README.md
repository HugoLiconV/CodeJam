# CodeJam-JavaScript
Solution for code jam problems using Javascript.

* Oversized Pancake Flipper 

* Tidy Numbers

* Bathroom Stalls

* Fashion Show

## Installation

```
cd JavaScript
```
```
npm install
```

## Execute 
```
cd JavaScript
```
Then:
 
Oversized Pancake Flipper 
```
npm run pancake
```
Tidy Numbers
```
npm run tidy
```
Bathroom Stalls
```
npm run bathroom
```
Fashion Show
```
npm run fashion
```
